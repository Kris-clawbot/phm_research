# PHM Hybrid Research Dashboard

Streamlit MVP dashboard that syncs papers from OpenAlex and displays a searchable table + detail view.

## Run

```bash
cd app
python3 -m streamlit run dashboard.py --server.address 0.0.0.0 --server.port 3001 --server.headless true
```

## Tests

```bash
pytest -q
```
